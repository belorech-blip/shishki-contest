# КП «ШИШКИ» — конкурс риелторов

Проект для акции риелторов: регистрация, вход, личный кабинет, админ-панель и PHP backend.

## Структура

```
backend/shishki_api/  PHP API для хостинга reg.ru
tilda/                HTML-код блоков T123 для Tilda
sql/                  SQL-схема без персональных данных
docs/                 Документация по URL и хостингу
assets/               Ссылки на логотипы и материалы
```

## Важно

`backend/shishki_api/config.php` не хранится в GitHub. Используйте `config.example.php` как шаблон и создавайте реальный `config.php` только на хостинге.
